Install Flic app in phone. Connect Flic button with the Flick app following the instructions in the app. Once done, let's setup the Flic app to make it work as a Talisman device:

Step 1: In the Flick app, click on the button to start configuring it:

Step 2: Next you will configure each interaction. Click on the “+” button for Click

Step 3: Scroll until the end and click on “Advanced”. It might seem like nothing happened, but make sure to scroll down, you will see the advanced options.

Step 4: From the options choose “Internet Request”

Step 5: In this screen you will have to follow several steps:

  • Type the following URL in the text field for URL: http://t.bounceme.net:43186/button
  • Select “PUT” from the options

  • In the body text field enter a number, depending which click type you are configuring:

    • For “click” enter the number “1”
    • For “double click” enter the number “2”
    • For “hold” enter the number 3 as seen here

  • In the “Key” field type “user”
  • In the “Value” field type your user name
  • Click “insert”

Step 6:

  • Make sure that you can see below the “Insert” button a gray field that shows that it has stored correctly your user name. If you can’t see this please make sure to repeat points 4 to 6 of Step 5.
  • Click done

Step 7: Repeat steps 2 to 6 in for “Double Click” and “Hold” Clicks.

The only difference is the value for the “Body” Field in Step 5:

In the body text field enter a number, depending which click type you are configuring:

  • For “click” enter the number “1”
  • For “double click” enter the number “2”
  • For “hold” enter the number 3 as seen here
